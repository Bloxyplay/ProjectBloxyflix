BloxyFlix
BloxyFlix is a free educational digital archive dedicated to the preservation and accessibility of North Korean cinema, television broadcasts, cultural performances, and related media.
The platform provides structured access to films, news broadcasts, live television feeds, and archival content for educational, research, and cultural awareness purposes only.
Official Site: https://bloxyflix.netlify.app

Mission Statement
BloxyFlix exists to preserve and make available primary source materials from North Korean state media in an organized, accessible, and responsible manner. All content is presented without alteration and is intended solely for academic and cultural study.
The platform operates on the principles of transparency, privacy, and accessibility. It remains completely free of charge, with no advertising or user tracking.
Technical Overview
Built with clean HTML, CSS, and vanilla JavaScript
Hosted on Netlify with GitHub as the source repository
